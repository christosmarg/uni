# Execution steps

* ```program_name``` = program's name
* ```full_path/``` = full path
* ```$``` = terminal command (__don't__ copy it)

## Linux

```shell
$ cd full_path/assignment-1.4-functions/program_name
$ make
$ make run
$ make clean #optional
```

## Windows

__Requirements:__ MinGW gcc compiler

In CMD, do the following:

```bat
cd full_path/assignment-1.4-functions/program_name
make
make run
make clean
```
